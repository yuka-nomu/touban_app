\# 登校班当番表作成アプリ 機能仕様書 v1.0



\---



\# 1. 概要



本書は「登校班当番表作成アプリ」の機能仕様を定義する。



対象機能：



\* 名簿管理

\* 当番表生成

\* カレンダー編集

\* ドラッグ並び替え

\* イベント注釈

\* 画像出力

\* ローカル保存

\# 5. 当番表生成機能



\---



\# 5.1 概要



対象月の当番表を自動生成する。



\---



\# 5.2 入力仕様



| 項目   | 内容 |

| ---- | -- |

| 対象年月 | 必須 |

| 開始人物 | 必須 |



\---



\# 5.3 自動生成ルール



\---



\## 対象日



| 条件    | 初期状態 |

| ----- | ---- |

| 月曜〜金曜 | ON   |

| 土曜    | OFF  |

| 日曜    | OFF  |

| 日本祝日  | OFF  |



\---



\## 当番配置ルール



\* 名簿順に配置する

\* 開始人物から開始する

\* ON日のみ配置する

\* 1日1人とする

\* 空欄日は作成しない

\* 自動配置は生成時のみ実行する
