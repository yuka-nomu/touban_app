\# 登校班当番表作成アプリ 画面設計書 v1.0



\---



\# 1. 画面一覧



| 画面ID | 画面名       | 概要               |

| ---- | --------- | ---------------- |

| S-01 | ホーム画面     | 対象月・開始人物選択、当番表生成 |

\# 4. S-01 ホーム画面



\---



\## 4.1 目的



当番表生成条件を設定し、カレンダー編集画面へ遷移する。



\---



\## 4.2 ワイヤーフレーム



```text

┌────────────────┐

│ 登校班当番表作成 │

├────────────────┤

│ 対象月           │

│ \[ 2026年5月 ▼ ] │

│                │

│ 開始人物         │

│ \[ 山田 ▼ ]      │

│                │

│ \[ 当番表生成 ]   │

│                │

│ \[ 設定 ]        │

└────────────────┘

```



\---



\## 4.3 UI項目



| UI項目   | 種別             | 内容      |

| ------ | -------------- | ------- |

| 対象月選択  | DropdownButton | 年月選択    |

| 開始人物選択 | DropdownButton | 名簿から選択  |

| 当番表生成  | ElevatedButton | カレンダー生成 |

| 設定     | OutlinedButton | 設定画面へ遷移 |



\---



\## 4.4 動作仕様



| 操作      | 動作                  |

| ------- | ------------------- |

| 当番表生成押下 | 当番表を生成しカレンダー編集画面へ遷移 |

| 設定押下    | 設定画面へ遷移             |

\# 5.3 カレンダー仕様



\---



\## 表示ルール



| 項目        | 内容       |

| --------- | -------- |

| カレンダー開始曜日 | 日曜開始     |

| 表示対象      | 当月のみ     |

| 土曜        | 日付を青文字表示 |

| 日曜        | 日付を赤文字表示 |

| OFF日      | グレーアウト表示 |

| 1日表示      | 1セル固定    |



\---



\## セル構成



```text

┌────────┐

│ 日付    │

│ 当番名  │

│ 注釈    │

└────────┘

```



\---



\## セルレイアウト仕様



| 項目    | 内容    |

| ----- | ----- |

| セル高さ  | 72dp  |

| セル幅   | 7分割均等 |

| 日付位置  | 左上固定  |

| 当番名位置 | 中央    |

| 注釈位置  | 下部    |

| 注釈行数  | 1行固定  |

| 名前行数  | 1行固定  |

| セル余白  | 4dp   |



\---



\## 名前表示仕様



| 項目   | 内容    |

| ---- | ----- |

| 長い名前 | 自動縮小  |

| 折返し  | 不可    |

| 省略記号 | 使用しない |



\---

\## 編集可能項目

\---



\# 5.5 ドラッグ仕様



\---



\## ドラッグ対象



| 項目  | 内容   |

| --- | ---- |

| 当番名 | 移動対象 |

| 注釈  | 固定   |

| 日付  | 固定   |



\---



\## ドラッグ動作



ドラッグ方式は「挿入スライド方式」とする。



\---



\## 動作例



変更前



```text

5/1 A

5/2 B

5/3 C

```



Aを5/3へ移動



変更後



```text

5/1 B

5/2 C

5/3 A

```

\# 6. S-03 設定画面



\---



\## 6.1 目的



名簿管理および初期設定を行う。



\---



\## 6.2 ワイヤーフレーム



```text

┌────────────────┐

│ ← 設定          │

├────────────────┤

│ 名簿管理         │

├────────────────┤

│ 1. 山田          │

│ 2. 田中          │

│ 3. 鈴木          │

│                │

│ \[+ 追加]         │

│                │

│ ドラッグで並替え │

└────────────────┘

```



\---



\## 6.3 UI項目



| UI項目  | 種別                   | 内容    |

| ----- | -------------------- | ----- |

| 名簿一覧  | ListView             | 登録順表示 |

| 追加ボタン | FloatingActionButton | 名前追加  |

| 並び替え  | ReorderableListView  | 順番変更  |



\---



\## 6.4 動作仕様



| 操作   | 動作          |

| ---- | ----------- |

| 追加   | 名前入力ダイアログ表示 |

| 削除   | スワイプ削除      |

| ドラッグ | 並び順変更       |



\---



\# 7. 共通カラー仕様



| 用途     | 内容          |

| ------ | ----------- |

| 土曜文字   | 青           |

| 日曜文字   | 赤           |

| OFF日背景 | グレー         |

| 通常背景   | 白           |

| ボタン    | Material3標準 |



\---



\# 8. UI制約事項



| 項目      | 内容               |

| ------- | ---------------- |

| カレンダー画面 | 編集作業を1画面で完結する    |

| 操作数     | 最小化する            |

| 画面遷移    | 必要最低限とする         |

| セルサイズ   | 月全体が1画面に収まることを優先 |

| 横スクロール  | 使用しない            |
